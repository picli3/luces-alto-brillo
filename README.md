# luces-alto-brillo

Driver para luces de alto brillo basado en **MC34063**, orientado a conversión y control de potencia para LEDs o cargas luminosas exigentes.

## Propósito del proyecto
El diseño busca generar una etapa compacta de alimentación/control para iluminación de mayor intensidad, usando una topología conmutada clásica y fácil de prototipar.

## Qué incluye
- esquemático del driver con MC34063
- PCB en KiCad
- base reutilizable para pruebas de iluminación y conversión DC-DC
